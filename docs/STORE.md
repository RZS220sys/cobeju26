# Store brief

## Short description

Carry the last lantern into a drowned archive. Fight memories that learned hunger, build a new combat language each descent, and decide whether a painful past should be preserved, released, or rewritten.

## Audience

Players who value readable action-roguelite combat, melancholy science-fantasy mystery, build experimentation, and choices without a labeled “good” answer.

## Features

- Responsive 3D casting, invulnerable slip, and crowd-control resonance.
- Seeded memory islands, five Tide conditions, three enemy families, elite variants, and a multi-pattern boss.
- Eleven stackable annotations plus permanent lantern workshop progression.
- 24 interconnected archive records, four pivotal verdicts, and three replayable conclusions.
- Story/Standard/Severe intensity; keyboard/mouse and controller parity.
- Offline, single-player, no telemetry, no ads, no microtransactions.

## Content notes

Themes of grief, drowning, memory loss, institutional harm, and identity. Stylized non-graphic combat; enemies disperse into light fragments. Brief screen shake can occur on damage.

## PC requirements

- Windows 10/11, 64-bit
- Dual-core 2.5 GHz CPU
- 4 GB RAM
- OpenGL 3.3-compatible GPU with 1 GB VRAM
- 150 MB storage
- 1280×720 display
